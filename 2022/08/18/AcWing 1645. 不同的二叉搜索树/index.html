题目

给定一个整数 ，求以 为节点组成的二叉搜索树有多少种？

结果对 取模后输出。

输入格式

共一行，包含一个整数 。

输出格式

输出一个整数，表示对 取模后的结果。

数据范围

输入样例：

3

输出样例：

5

样例解释

当 时, 一共有 种不同结构的二叉搜索树:

1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
